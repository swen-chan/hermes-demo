# Hermes Demo

用 Hermes Agent 准备一场关于 Hermes Agent 的轻量分享，并把准备过程本身作为 Meta-Demo 展示。

## 目标

- 在 OPC 大理 24h 创客松活动中做一场关于 Hermes Agent 搭建与使用的分享
- 整个准备流程由 Hermes Agent 驱动：从大纲、逐字稿、slides、分发物料到风险检查
- 分享本身就是最好的 demo — "我用 Hermes 准备了这场关于 Hermes 的分享"

## 工作区结构

| 文件 | 用途 |
|---|---|
| `00-context.md` | 活动背景、受众画像、分享约束 |
| `01-talk-outline.md` | 分享大纲与时间分配 |
| `02-slides.md` | Slides 逐页内容 |
| `03-hermes-guide.md` | Hermes 搭建指南（可独立分发） |
| `04-distribution.md` | 分发策略：推文、公众号、即刻等 |
| `05-video-script.md` | 录屏/视频脚本 |
| `06-final-pack.md` | 最终打包清单 |
| `07-risk-checklist.md` | 现场风险检查清单 |
| `assets/` | 录屏、截图、PPT 导出图等素材 |
